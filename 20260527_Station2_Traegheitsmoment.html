Trägheitsmoment, Rotationsenergie & Steinerscher Satz

📋 Situation

Auf einem Spielplatz steht ein einfaches Kinderkarussell: eine drehbare Metallscheibe (Vollzylinder), die von außen angeschoben wird.

Masse der Scheibe:mS = 200 kg
Radius:R = 1,5 m
Drehzahl:n = 12 min⁻¹
✓
Aufgabe 2a – Trägheitsmoment der Scheibe

Berechnen Sie das Trägheitsmoment JS der Karussellscheibe.

Die Scheibe ist ein Vollzylinder. Welche Formel gilt für einen Vollzylinder?
JS = ½ · m · R² = ½ · 200 · 1,5² = …
JS = ½ · 200 · 2,25 = 225 kg·m²
✓
Aufgabe 2b – Rotationsenergie

Berechnen Sie die Winkelgeschwindigkeit ω und die Rotationsenergie Erot des drehenden Karussells.

Vorsicht bei der Drehzahl! n = 12 min⁻¹ muss erst in s⁻¹ umgerechnet werden, bevor Sie ω = 2π·n verwenden.
n = 12/60 s⁻¹ = 0,2 s⁻¹
ω = 2π · 0,2 = …
Erot = ½ · J · ω²
ω = 2π · 0,2 = 1,257 rad/s
Erot = ½ · 225 · 1,257² = ½ · 225 · 1,58 ≈ 178 J
✓
Aufgabe 2c – Kind am Rand

Ein Kind (mK = 30 kg) setzt sich auf den äußersten Rand der Scheibe.

a) Berechnen Sie den zusätzlichen Beitrag des Kindes zum Gesamt-Trägheitsmoment (Punktmasse im Abstand R).

b) Bestimmen Sie das neue Gesamt-Trägheitsmoment Jges.

c) Um wie viel Prozent hat sich J vergrößert?

Das Kind sitzt am Rand (Abstand R) – behandeln Sie es als Punktmasse. Addieren Sie den Beitrag zum J der Scheibe.
JKind = mK · R² = 30 · 1,5² = …
Jges = JS + JKind
JKind = 30 · 2,25 = 67,5 kg·m²
Jges = 225 + 67,5 = 292,5 kg·m²

Zunahme: 67,5 / 225 · 100% = 30 %
✓
Aufgabe 2d – Versetzter Drehpunkt (Steiner)

Durch Verschleiß hat sich die Lagerachse des Karussells um d = 0,10 m vom Mittelpunkt der Scheibe verschoben.

a) Berechnen Sie das Trägheitsmoment der Scheibe bezüglich der neuen Achse mithilfe des Steinerschen Satzes.

b) Vergleichen Sie mit dem ursprünglichen Wert. Ist der Unterschied groß oder klein? Begründen Sie.

Der Steinersche Satz: J = JS + m · d². JS ist das Ergebnis aus a), d = 0,10 m.
Jneu = 225 + 200 · 0,10² = 225 + 200 · 0,01 = …
Jneu = 225 + 2 = 227 kg·m²

Zunahme: 2 / 225 · 100% ≈ 0,9 % – kaum spürbar, weil d ≪ R.
✓
Aufgabe 2e – Nachdenken: Kind rutscht zur Mitte

Das Karussell dreht sich mit dem Kind am Rand. Das Kind rutscht nun langsam zur Mitte der Scheibe.

Beschreiben Sie qualitativ (ohne Rechnung), was mit der Drehgeschwindigkeit passiert und begründen Sie Ihre Antwort.

Hinweis: Überlegen Sie, welche physikalische Größe bei diesem Vorgang erhalten bleibt.

Welche Erhaltungsgröße gilt, wenn keine äußeren Drehmomente wirken?
Der Drehimpuls L = J · ω bleibt erhalten.
Wenn J kleiner wird (Kind zur Mitte), muss ω …
Der Drehimpuls L = J · ω ist erhalten (kein äußeres Drehmoment).

Kind rutscht zur Mitte → J wird kleiner → ω muss größer werden → das Karussell dreht sich schneller.

Das ist der Pirouetteneffekt – genau wie eine Eiskunstläuferin, die die Arme anzieht!

📎 Formelsammlung

JVollzylinder = ½·m·R²    JPunktmasse = m·r²    J = JS + m·d² (Steiner)
ω = 2π·n    Erot = ½·J·ω²    L = J·ω (Drehimpuls)
